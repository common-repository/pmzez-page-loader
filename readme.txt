== Description ==

# PmZez Page Demo loader by http://www.pmzez.com/plugins/pmzez-page-loader

This plugin will Slim progress bars for Ajax'y applications. Inspired by Google, YouTube, and Medium.

Go to ***"Settings >> PmZez Page Loader"*** -on the bottom and you can change loader color using color picker.

Plugin Features

* Slim progress bar.
* Ajax'y applications. 
* Dynamically call jQuery Library.
* Fully Responsive. 
* Mobile supported. 
* Very Lightweight. Only 6.5KB.
& many More

Live Preview: http://www.pmzez.com/plugins/pmzez-page-loader

== Installation ==

* Install it as a regular WordPress plugin.

* After Installed this plugin will work properly.

* Go to Settings >> PmZez Page Loader -on the bottom and you can change loader color.

* If you are logged in WordPress Dashboard , logged out and then check - After refresh your browser Ajex loader will appear on the top of the page .

* Enjoy 

== Frequently asked questions ==


= Loader not show on top of the page? =
If you are logged in WordPress Dashboard , logged out and then check - After refresh your browser Ajex loader will appear on the top of the page . 

= Can I change color of loader? =
Yes you can . Go to Settings >> PmZez Page Loader -on the bottom and you can change loader color using colorpicker.
